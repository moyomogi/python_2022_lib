# algebra/calc_rect_area

長方形の面積 $height \times width$ を求めます。計算量は $O(1)$ です。

[View on GitHub](https://github.com/moyomogi/python_2022_lib/blob/master/lib/algebra/calc_rect_area.py)

```py
def calc_rect_area(height, width):
    return height * width
```
